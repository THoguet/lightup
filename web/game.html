Lightup-07C

Placer des ampoules pour éclairer toutes les cases blanche. Une ampoule éclaire sa propre case ainsi toutes les case de sa ligne et de sa colonne sauf si la lumière est bloquée par un mur. Les ampoules ne doivent pas s'éclairer entre elles. Chaque mur avec un chiffre doit avoir cette exacte quantité d'ampoule collée(s) à lui.
Pour placer une ampoule il faut faire clique gauche, pour placer une marque c'est clique droit, vous ne pouvez pas placer d'ampoule sur les marques.
Il est possible de ne jouer qu'avec le clavier: flèches directionnelles pour deplacer le curseur, b pour poser une ampoule, m une marque, z pour undo, y pour redo, r pour restart, n pour new game, s pour solve, w pour sauvegarder, l pour charger une partie, k pour partager.
